C언어 STM32 GitHub Range of Motion

Kotlin C언어 STM32 Modbus Protocol GitHub Monitoring System

LoRa 통신·LCD 제어·안드로이드 앱을 결합해 사용자의 운동 상태를 실시간 모니터링할 수 있는 시스템을 구현했습니다. 연구원들과 협업하여 통신 프로토콜 문제를 해결했고, 데이터 시각화 및 기록 관리 기능을 확보했습니다. 성과는 한국정보기술학회 추계 학술대회 논문집에 게재되었습니다.

지능형 열 감지기를 활용한 실시간 위치 추적 화재 감지 시스템 설계 25.03~25.08

C언어 STM32 Modbus Protocol Power Line Communication GitHub

PLC(Power Line Communication) 기반으로 저비용·고효율의 위치 추적 화재 감지기를 개발했습니다. ADC 온도 측정, UART 통신, 지능형 알고리즘을 결합해 기존 대비 빠르고 정확한 감지를 실현했습니다. LCD 표시기를 통해 화재 발생 위치를 실시간으로 확인할 수 있는 구조를 설계했습니다.
